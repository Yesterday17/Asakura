## 特色：

<details>
<summary>文字介绍</summary>

```
- 功能类：
    - 主页随机图多端分离（支持API）
    - 内嵌提示块（短代码）
    - 特效资源云端化
    - 原生集成升级功能
    - 强大的色彩控制功能
    - 多种全局定制色彩图标
    - 网页预加载动画
    - 强大的字体设置功能
    - 可选择的夜间模式
- 特效类：
    - 樱花飘落特效
    - 波浪特效
    - 看板娘 Live2D
    - 首页下拉箭头
    - 脚页一言（随机话语API）
    - 首页一言打字效果
    - logo音符触动效果
    - logo特效
    - 邮件模板自定义

```
</details>
