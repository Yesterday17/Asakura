<h1 align="center"> 「アサクラ」(o・∇・o)</h1>

## 如遇问题

- 本主题不提供任何技术支持
- 如有需求请自由 Fork 修改

## 下载：

本主题的建议安装方式为直接 `git clone`，但你也可以下载 [Release 版](https://github.com/Yeterday17/Asakura/releases/latest) 。
